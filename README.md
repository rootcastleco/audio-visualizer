# Audio Visualizer

An elegant and interactive audio visualizer built using modern web technologies.

## Features

- **Audio Spectrum Visualization**: Real-time visualization for audio files or live microphone input.
- **Customizable Design**: Choose your colors, adjust sensitivity, and switch between multiple visualization modes.
- **Modern and Responsive**: Built to look great on devices of all sizes.
- **Interactive UI**: Intuitive controls for users to load audio, adjust settings, and toggle between different modes.

## Usage

1. **Load an Audio File**:
   - Click on the "Choose Audio" button to select a local audio file.
2. **Use a Microphone**:
   - Select the "Microphone" option for live audio visualization (requires permission).
3. **Switch Modes**:
   - Experiment with various modes like Bars, Waves, Circular, and Particles.
4. **Customize Settings**:
   - Change visualization colors, volume levels, and sensitivity to suit your preference.

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/rootcastleco/audio-visualizer.git
   ```
2. Navigate into the project folder:
   ```bash
   cd audio-visualizer
   ```
3. Open the `index.html` file in your favorite browser:
   ```bash
   open index.html
   ```
   or double-click the file directly.
